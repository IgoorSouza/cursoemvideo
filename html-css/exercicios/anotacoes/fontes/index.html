Anatomia do Tipo

OBS.: Todas as fontes são criadas a partir da definição de sua letra x minúscula.
OBS2.: Glifo = caractere. Fonte = conjunto de caracteres que seguem um padrão de componentes anatômicos.
Uma mesma letra pode formar 2 glifos diferentes, um sendo sua versão maiúscula e o outro a versão minúscula (Ex.: "A" e "a").

Serifa

Serve para "guiar" o cérebro na leitura através de uma linha imaginária.

Família Tipográfica

São as variações das letras de uma fonte.
Geralmente, toda fonte tem pelo menos as variações "normal" e "negrito", com algumas exceções onde há apenas a normal.

Categoria de Fontes

Sans-serif = sem serifa. Recomendada em textos longos para impressão em sites.
Display: fontes comemorativas; não se preocupam em obedecer regras anatômicas.
