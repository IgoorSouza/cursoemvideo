Código-fonte / Pré-formatação

A tag <code> deixa as letras monoespaçadas (mesmo espaço para todas as letras).

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

A tag <pre> mantém a forma da linha de texto da maneira como foi escrita.

Sem a tag <pre>:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Com a tag <pre>:

      
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é par')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa')
